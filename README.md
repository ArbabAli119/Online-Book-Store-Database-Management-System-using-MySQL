# 📚 Online Book Store Database Management System

## 📖 Overview
This project demonstrates the design and implementation of an Online Book Store Database using MySQL. It manages books, customers, and orders while performing inventory tracking, sales analysis, and customer management through SQL queries.

## 🚀 Features
- Database schema design using MySQL
- Books, Customers, and Orders management
- CSV data import and processing
- Inventory and stock tracking
- Sales and revenue analysis
- SQL joins, filtering, sorting, and aggregation
- Real-world database query practice

## 🛠️ Tech Stack
- MySQL
- SQL
- CSV

## 🗄️ Database Tables
### Books
Stores book details such as:
- Book ID
- Title
- Author
- Genre
- Price
- Stock

### Customers
Stores customer information such as:
- Customer ID
- Name
- Email
- City
- Country

### Orders
Stores order details such as:
- Order ID
- Customer ID
- Book ID
- Quantity
- Order Date

## 📊 SQL Operations Performed
- Data Retrieval
- Filtering Records
- Sorting Data
- Aggregate Functions
- JOIN Operations
- Revenue Analysis
- Inventory Management
- Customer Insights

## 📂 Project Files
- `OnlineBookStore.sql`

- `Customers.csv`
- `Orders.csv`
